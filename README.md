# Minitalk
The purpose of this project is to code a small data exchange program between a *client* and a *server* using only two UNIX signals (SIGUSR1 and SIGUSR2).
Allowed functions:
```
◦ write
◦ ft_printf and any equivalent YOU coded
◦ signal
◦ sigemptyset
◦ sigaddset
◦ sigaction
◦ kill
◦ getpid
◦ malloc
◦ free
◦ pause
◦ sleep
◦ usleep
◦ exit
```
